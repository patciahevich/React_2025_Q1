# STAR WARS API (React Learning Project)

## 📍 Overview

This project is a React application aimed at learning and exploring React features. It serves as a practice ground for understanding React concepts, state management, component composition, and API interactions.

## 🔎 Technologies Used

- React: Frontend library for building UI

- TypeScript: Strongly typed JavaScript for better code quality

- SASS: CSS preprocessor for improved styling

## 🔗 Installation

1. Clone the repository:

   `git clone https://github.com/patciahevich/React_2025_Q1.git`

2. Go to the necessary branch:

   `git checkout class-components`

3. Go to the project folder:

   `cd react-project`

4. Install dependencies:

   `npm install`

5. Start the development server:

   `npm run dev`

6. Run test:

   `npm run test`

### ❗️ N.B.: If the is no access to the API (star wars api), please use VPN to test my app. Thank you 😇
